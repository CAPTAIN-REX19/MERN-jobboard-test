# MERN Дошка Вакансій (Job Board)

Повноцінний веб-застосунок для пошуку та розміщення вакансій, створений за допомогою стеку MERN (MongoDB, Express, React, Node.js).

## Можливості

- Реєстрація та авторизація користувачів за допомогою JWT
- Створення, перегляд, оновлення та видалення заявок на роботу (CRUD)
- Фільтрація за статусом, типом зайнятості, пошук за посадою
- Сортування та пагінація результатів
- Захищені маршрути панелі керування (dashboard)

---

## Структура проекту

```
mern-job-board/
├── client/      # Фронтенд на React + Vite
├── server/      # Бекенд на Express + Mongoose
├── package.json # Кореневі скрипти (concurrently)
└── .env.example
```

---

## Змінні середовища (Environment Variables)

Створіть файл `.env` у папці `server/` на основі `.env` у глобалі:

| Змінна       | Опис                                         |
|--------------|----------------------------------------------|
| `MONGO_URI`  | Рядок підключення до MongoDB Atlas           |
| `JWT_SECRET` | Секретний ключ для підпису JWT токенів       |
| `JWT_LIFETIME` | Час дії токена (наприклад, `1d`)             |
| `PORT`       | Порт сервера (за замовчуванням `5000`)       |

## Встановлення та запуск

### 1. Клонування та встановлення залежностей

```bash
git clone https://github.com/CAPTAIN-REX19/MERN-jobboard-test.git
cd MERN-jobboard-test
npm install          # встановлює кореневі залежності (concurrently)
cd server && npm install
cd ../client && npm install
```

Або скористайтесь швидкою командою:

```bash
npm run install-all
```

### 2. Налаштування середовища

```bash
cp .env server/.env
```

### 3. Запуск у режимі розробки

```bash
npm run dev
```

Ця команда одночасно запускає Express API (порт 5000) та dev-сервер Vite (порт 5173).  
Vite-проксі перенаправляє запити `/api` на `http://localhost:5000`.

### 4. Збірка для продакшену та розгортання (Render)

Оскільки бекенд налаштовано для роздачі статичних файлів фронтенду, ви можете розгорнути проєкт як один єдиний Web Service.

**Build Command:**
```bash
npm run install-all && cd client && npm run build
```

**Start Command:**
```bash
cd server && npm start
```

## API Ендпоінти

### Авторизація — `/api/v1/auth`

| Метод  | Шлях        | Опис                   | Авторизація |
|--------|-------------|------------------------|-------------|
| POST   | /register   | Реєстрація нового користувача | Ні          |
| POST   | /login      | Вхід, повертає JWT     | Ні          |

### Вакансії (Jobs) — `/api/v1/jobs`

| Метод  | Шлях   | Опис                       | Авторизація |
|--------|--------|----------------------------|-------------|
| GET    | /      | Отримати всі вакансії (з фільтрами) | Так         |
| POST   | /      | Створити нову вакансію     | Так         |
| PATCH  | /:id   | Оновити вакансію           | Так         |
| DELETE | /:id   | Видалити вакансію          | Так         |

### Параметри запиту (GET /api/v1/jobs)

| Параметр | Значення                            | За замовчуванням |
|----------|-------------------------------------|------------------|
| status   | pending, interview, declined, all   | all              |
| jobType  | full-time, part-time, remote, all   | all              |
| sort     | latest, oldest, a-z, z-a            | latest           |
| search   | рядок (пошук за полем посади)       | —                |
| page     | число                               | 1                |
| limit    | число                               | 10               |
